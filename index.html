About Events Code of Conduct

闘魂Elixirとは

Elixir / Phoenix を中心に学び、創り、発表するコミュニティです。

私たちの目的：アイデアの炎をコードに変え、挑む仲間が集い、読み、書き、創る場所を提供すること。

私たちの価値：

  • 学びの共有 - コードと文章で記録し、成果を世界へ届ける
  • 包括性 - 初心者歓迎。経験者は設計とレビューで火力を足す
  • 継続的成長 - 闘魂とは己に打ち克つこと。闘いを通じて己の魂を磨く
  • 品質重視 - 速度より品位。丁寧なものづくりを大切にする

参加方法

  • GitHub の README を読み、PR で自己紹介。小さく始めて早く出す。
  • Qiita / Zenn で学びを公開。リンクを追加。
  • 月1 LT で成果共有。5分でも良い。挑む頻度が火力。

行動規範

闘魂Elixirコミュニティでは、すべてのメンバーが安心して参加できる環境を大切にしています。

  • 尊重と誠実 - すべてのメンバーを尊重し、誠実なコミュニケーションを心がける
  • 学びの共有 - 知識や経験を積極的に共有し、共に成長する
  • 建設的な議論 - 誹謗中傷は禁止。建設的で前向きな議論を推進する
  • 知的財産の尊重 - ライセンスとクレジットを適切に守る
  • 品質重視 - 速度より品位。丁寧で質の高いものづくりを目指す

詳細な行動規範は GitHub リポジトリでご確認ください。
